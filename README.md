# CareerHub – Online Job Portal

## Project Overview

CareerHub is a web-based job portal designed to connect Job Seekers and Recruiters through a single platform. The system enables recruiters to post job opportunities and manage applications, while job seekers can search for jobs, view job details, and apply for suitable positions.

The platform aims to simplify the recruitment process by providing an easy-to-use interface for both recruiters and job seekers.

---

## Project Objectives

* Provide a centralized platform for job seekers and recruiters.
* Allow recruiters to post and manage job vacancies.
* Enable job seekers to search and apply for jobs.
* Streamline the recruitment and application process.
* Improve communication between recruiters and candidates.

---

## Key Features

### For Job Seekers

* User Registration and Login
* Profile Management
* Job Search and Filtering
* View Job Details
* Apply for Jobs
* Track Applied Jobs
* Upload Resume/Profile Information

### For Recruiters

* Recruiter Registration and Login
* Create and Manage Job Listings
* View Candidate Applications
* Manage Posted Jobs
* Candidate Screening and Selection

### General Features

* Secure Authentication System
* Role-Based Access Control
* Responsive User Interface
* Media and File Upload Support
* Email Notifications
* Dashboard for Users

## System Modules

### Authentication Module

Handles registration, login, logout, and user authentication.

### Job Management Module

Allows recruiters to create, update, delete, and manage job postings.

### Application Management Module

Enables job seekers to apply for jobs and recruiters to review applications.

### Profile Management Module

Allows users to manage personal and professional details.
